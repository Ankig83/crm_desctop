# Руководство пользователя — CRM Desktop

**Версия программы:** 1.0  
**Последнее обновление:** апрель 2026

---

## Что это за программа?

CRM Desktop — это программа для менеджеров отдела продаж. Она помогает:

- хранить базу клиентов и товаров;
- назначать акции и скидки на товары;
- быстро рассчитывать стоимость заказа;
- выгружать готовый файл `RUS.xlsx` для автоматического попадания заказа в 1С;
- отправлять расчёт на e-mail клиенту.

Все данные хранятся прямо на компьютере — никакой интернет не нужен.

---

## Запуск программы

Откройте файл **CRM_Desktop.exe** (или ярлык на рабочем столе).  
База данных хранится здесь и никуда не теряется при обновлении программы:
```
C:\Users\<Ваше имя>\AppData\Local\CRM_Desktop\crm.db
```

## Вкладки программы

### 1. Клиенты

Здесь хранится список всех клиентов.

**Как найти клиента:**  
Начните вводить название или ИНН в строку поиска — список сразу отфильтруется.

**Как добавить клиента вручную:**
1. Нажмите кнопку **«+ Новый клиент»**.
2. Заполните поля справа: название, ИНН, контакты, адрес.
3. Данные сохраняются автоматически — кнопку «Сохранить» нажимать не нужно.

**Поля клиента:**

| Поле | Пояснение |
|------|-----------|
| Название | Полное название организации |
| ИНН | 10 или 12 цифр (без пробелов) |
| Тип клиента | Дистрибутор / Оптовик / Торговая сеть / Обычный |
| Контакты | Телефоны — каждый с новой строки |
| Адреса | Адреса — каждый с новой строки |
| Пункты разгрузки | Адреса доставки — каждый с новой строки |
| Новый клиент | Флажок: снимите, когда обработали нового клиента |
| Грузополучатель | Если нужна отдельная организация для доставки |

**Тип клиента** влияет на размер скидки в расчёте.

---

### 2. Товары

Список всех товаров с ценами.

**Как найти товар:**  
Введите название или артикул в строку поиска.

**Как добавить товар:**  
Нажмите **«+ Добавить товар»**, заполните поля. Основное:

| Поле | Пояснение |
|------|-----------|
| Артикул | Внешний ID товара (совпадает с 1С) |
| Наименование | Название товара |
| Базовая цена | Цена без скидок (рублей) |
| Кол-во в коробке | Штук в одной коробке |
| Цена за штуку | Рассчитывается автоматически |

Логистические поля (вес, объём, размеры) нужны для выгрузки в 1С — заполняйте при наличии данных.

---

### 3. Акции

Здесь настраиваются скидки и бонусы для каждого товара.

**Правила:**
- На один товар — только одна акция.
- Скидка задаётся в процентах.
- Акция работает только в заданный период (обязательно укажите **дату начала** и **дату окончания** в формате **ДД.ММ.ГГГГ**).

**Как добавить акцию:**
1. Выберите товар из списка слева.
2. Нажмите **«Новая акция»**.
3. Заполните поля справа.
4. Нажмите **«Сохранить акцию»**.

**Что можно настроить в акции:**

| Блок | Что делает |
|------|-----------|
| Скидки за предоплату | Если клиент вносит X% предоплаты — даём скидку Y% |
| Скидки за объём | Если заказывает больше 300 или 500 коробок — скидка |
| Продуктовая скидка | Скидка за «истекающий» товар (% или рублей) |
| Акционные бонусы | При покупке N коробок — M коробок бесплатно |

Бонусы: можно указать «бесплатно тот же товар» или «бесплатно другой товар» (указывается артикул через запятую).

---

### 4. Расчёт — главная вкладка

Здесь формируется заказ и выгружается файл для 1С.

**Шаг 1. Выберите клиента** — из выпадающего списка.  
**Шаг 2. Укажите даты** — дату расчёта и дату доставки.  
**Шаг 3. Укажите № заказа** — заполняется автоматически, можно исправить.  
**Шаг 4. Укажите предоплату** — если клиент готов платить авансом (0–100%).  
**Шаг 5. Добавьте товары** — кнопка **«+ Добавить товар»**, выберите нужное, укажите количество.

Программа автоматически:
- применит скидку за тип клиента;
- применит скидку за предоплату (если настроена в акции);
- применит скидку за объём;
- добавит бонусные коробки (если акция предусматривает).

**Кнопки экспорта:**

| Кнопка | Что делает |
|--------|-----------|
| Сформировать RUS.xlsx | Файл для автозагрузки в 1С. Имя файла: `RUS{номер}.xlsx` |
| Сохранить расчёт (PDF) | Красивый документ для клиента |
| Сохранить расчёт (TXT) | Простой текстовый файл |
| Отправить на e-mail | Отправляет расчёт письмом (нужно настроить почту) |
| Сохранить в историю | Сохраняет снимок расчёта в журнале |

> **Важно про RUS.xlsx:** после нажатия кнопки номер заказа увеличивается автоматически (1 → 2 → 3…). Если нужно переформировать тот же заказ — исправьте номер вручную перед экспортом.

## Меню «Файл»

| Пункт | Что делает |
|-------|-----------|
| Импорт клиентов (Excel) | Загрузить клиентов из файла `clients.xlsx` |
| Импорт товаров (Excel) | Загрузить товары из файла `products.xlsx` |
| Импорт акций (Excel) | Загрузить акции из файла `promotions.xlsx` |
| Экспорт клиентов | Выгрузить текущий список клиентов в Excel |
| Экспорт товаров | Выгрузить текущий список товаров в Excel |
| Экспорт акций | Выгрузить акции в Excel (со всеми настройками) |
| Резервная копия базы | Сохранить копию базы данных в папку |
| Настройки | Настройка почты SMTP для отправки e-mail |

---

## Импорт из Excel — требования к файлам

### clients.xlsx — обязательные колонки:
`ID клиента`, `Название`, `ИНН`

### products.xlsx — обязательные колонки:
`ID товара`, `Наименование`, `Базовая цена`

### promotions.xlsx — обязательные колонки:
`ID товара`, `Размер скидки`, `Дата начала`, `Дата окончания`

Формат дат везде: **ДД.ММ.ГГГГ** (например: `01.04.2026`)

> Если при импорте есть ошибки — программа покажет список проблемных строк. Исправьте файл и импортируйте заново.

## Резервное копирование

Рекомендуем делать резервную копию базы раз в неделю:  
**Файл → Резервная копия базы** → выберите папку (например, на флешке или в облаке).

## Настройка почты

**Файл → Настройки** → вкладка SMTP.

Заполните данные вашего почтового ящика (узнайте у системного администратора):
- Адрес сервера SMTP
- Порт (обычно 587 или 465)
- Логин и пароль
- Ваш e-mail (из которого будут отправляться письма)

## Часто задаваемые вопросы

**Q: Данные потеряются при обновлении программы?**  
A: Нет. База данных хранится отдельно от программы в папке `AppData\Local\CRM_Desktop`.

**Q: Можно работать без интернета?**  
A: Да, всё работает полностью офлайн, кроме отправки e-mail.

**Q: Что делать если программа не запускается?**  
A: Убедитесь, что на компьютере установлен Visual C++ Redistributable. Если не помогло — обратитесь к разработчику.

**Q: Можно ли изменить тип скидки с % на рубли?**  
A: Сейчас скидка задаётся только в процентах. Дополнительная скидка в рублях — в поле «Продуктовая скидка (руб)».

**Q: Файл RUS.xlsx не загружается в 1С?**  
A: Убедитесь, что в расчёте указана дата доставки, клиент с ИНН, и у всех товаров указан артикул (ID товара), совпадающий с 1С.
